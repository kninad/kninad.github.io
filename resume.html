Projects

Faster Inference for Chow-Liu Trees

Developed approximation algorithms for faster inference in Chow-Liu tree probabilistic graphical model

Focused on trying out sub-quadratic time variants for minimum weight spanning tree computation

Proposed approximations demonstrated competitive results against the optimal setting

Report and Code: github.com/kninad/clt-mlproj

Data Subset Selection

Studied optimization problem of training data subset selection with minimal impact on validation loss

Utilized gradient approximation scheme to show utility on logistic regression and neural network models

Part of Speech Tagger for Hindi

Designed a semi-supervised method for a Hindi PoS tagger by leveraging an existing English tagger

FastText word embeddings and a parallel corpus between the 2 languages were used for method

Showed performance gain of 12% on domain classification of Hindi docs by using inferred PoS labels

Report and Code: github.com/kninad/POStagging_Hindi

Marching Cubes

Coded a base version of marching cubes algorithm for 3D iso-surface extraction using OpenGL

Improved initial results via better representation for vertex normal by averaging over faces for a vertex

Report and Code: github.com/kninad/marching-cubes-graphics
