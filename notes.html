Creating this website

This website was created with the help of jemdoc. It is convinient to create the html pages through its markdown like syntax. GitHub pages were used for (free) hosting the website. Make sure that jemdoc is in the PATH:

sudo cp jemdoc.py /usr/bin/jemdoc
sudo chmod +x /usr/bin/jemdoc

I wrote a very simple bash script for the conversion (could also be done with a makefile).
